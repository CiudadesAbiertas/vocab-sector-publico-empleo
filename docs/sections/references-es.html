Se reutilizan principalmente cuatro vocabularios:

  • esadm, el vocabulario de territorio, para conocer de dónde es la oferta de empleo público.
  • schema.org, el vocabulario general de los buscadores de contenido web, para el documento boletín oficial y datos temporales.
  • eli, European Legislation Identifier, el vocabulario de la Unión Europea para los datos legislativos, para el documento boletín oficial.
  • Vocabulario skos para las taxonomías
